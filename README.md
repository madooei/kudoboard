# Kudoboard

A simplified clone of the Kudoboard website, built using React and TypeScript. Users can create and share virtual boards to celebrate special occasions, express gratitude, or commemorate milestones.

## Run locally

- Clone this repository.
- Navigate to the project directory.
- Install dependencies using `pnpm install`.
- Start the development server with `pnpm run dev`.
- Open your browser and navigate to `http://localhost:5173`.
